* Bitmain SDHCI controller (sdhci-bm)

This file documents differences between the core properties in mmc.txt
and the properties used by the sdhci-msm driver.

Required properties:
- compatible: Should be one of
        "bitmain,bm-emmc"
        "bitmain,bm-sd"
        "bitmain,bm-pldm-emmc"
        "bitmain,bm-pldm-sdcard"
        "bitmain,bm1880-emmc"
        "bitmain,bm1880-sd"
- reg: Base address and length of the register in the following order:
	- Host controller register map (required)
	- SD Core register map (required)
- interrupts: Should contain an interrupt-specifiers for the interrupts:
	- Host controller interrupt (required)
- pinctrl-names: Should contain only one value - "default".
- pinctrl-0: Should specify pin control groups used for this controller.
- clocks: A list of phandle + clock-specifier pairs for the clocks listed in clock-names.
- clock-names: Should contain the following:
	"core"	- SDC MMC clock (MCLK) (required)

Example:

	emmc:bm-emmc@4fffe000 {
		compatible = "bitmain,bm-emmc";
		reg = <0x0 0x4fffe000 0x0 0x1000>;
		reg-names = "core_mem";
		interrupts = <GIC_SPI 51 IRQ_TYPE_LEVEL_HIGH>;//83-32=51 emmc intr ,84 emmc_wakeup_intr,85 sdio_intr,86 sdio_wakeup_intr
		bus-width = <4>;
		non-removable;
		no-sdio;
		no-sd;
	};

	sd:bm-sd@4ffff000 {
		compatible = "bitmain,bm-sd";
		reg = <0x0 0x4ffff000 0x0 0x1000>;
		reg-names = "core_mem";
		interrupts = <GIC_SPI 53 IRQ_TYPE_LEVEL_HIGH>;//83-32=51 emmc intr ,84 emmc_wakeup_intr,85 sdio_intr,86 sdio_wakeup_intr
		bus-width = <4>;
		no-sdio;
		no-mmc;
		resets = <&rst RST_SDIO>;
		reset-names = "sdio";
	};
